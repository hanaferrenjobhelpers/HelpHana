## Run locally

To try the app in your browser, start a local web server from the `HelpHana` folder.

If you have Python 3 installed, run:

```powershell
Set-Location e:\Files\Projects\HelpHana\HelpHana
python -m http.server 5500
```

Then open:

```text
http://localhost:5500
```

To test from a phone on the same Wi-Fi network, replace `localhost` with your computer's local IP address, for example:

```text
http://192.168.1.10:5500
```

If you use VS Code, you can also use the Live Server extension and open `index.html` directly from the editor.
